CyberMobilite
=============

Ce dépôt correspond au dépôt de code du projet CyberMobilité. 

Contenu du répertoire
---------------------

La structure du répertoire est conforme à la [structure ModelScript](https://modelscript.readthedocs.io/en/latest/artefacts/index.html) mais un document Coda est disponible comme dépôt documentaire. 

* `/modeles.mdj` : fichier StarUML contenant tous les modèles UML.
* `/dev/` : code de l'application
* `/concepts/besoins/capture/` : intégralité des documents et données collectées durant la capture des besoins.
* `/CONTRINBUTORS.json` : liste de tous les participants du groupe,
* '/CREDITS.md' : liste de toutes les éléments réutilisés,

Le contenu des répertoires suivants ont été migrés vers Coda :

* `/bd`
* `/cu`
* `/ihm`
* `/participants`
* `/projets`
